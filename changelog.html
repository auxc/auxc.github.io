Introducing the first usable version of cobweb. The website has basic search functionality to browse the index, and a frontpage featuring all available tags.

The goal of the project is to create a place where a small group of friends can share and browse interesting bits of the broader internet. Development will focus on improving link sharing, search, organization and archiving features.

All the underlying tech is naive and expected to be replaced with something robust once I'm done experimenting with features. It is however in the spirit of this endeavor to be at least a little old fashioned. Expect a certain level of hacky whimsy that was lost in computing somewhere along the way moving on from Windows 98. Hence the name, cobweb. Still the world wide web, but it feels like visiting a quirky old attic instead.

Search

  1. Displays all items from the index that match the search terms.
  2. Spaces between input words are treated as logical AND. If any words do not match at all, an item is not considered a search result.
  3. Page order is based on partial match score calculated using Levenshtein Distance. Close matches in header and description text are valued more than ones in tags and hidden keywords.
  4. Search can be further narrowed down by specifying tags using the following syntax: tag1: tag2: tag3: (...)

Frontpage

  1. All tags present in the index are displayed as links on the frontpage when no search terms are provided.

Link bot

  1. Users access the bot through a private group chat.
  2. Links and metadata are picked up from the conversation and inserted into the index.
